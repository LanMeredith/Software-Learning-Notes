# 操作

要添加打印机，打开电脑的 **打印机和扫描机** 进行添加

![](assets/Pasted%20image%2020250228151327.png)

选择 **按名称选择共享打印机**，输入打印机名称，格式为 `\\计算机名\共享打印机名称`

![](assets/Pasted%20image%2020250228151510.png)

当提示输入用户名和密码时，输入共享打印机所在电脑的用户名和密码即可

# 提供的凭证不足

进入 **控制面板** 点击 **凭据管理器** → **管理Windows凭据**

![](assets/Pasted%20image%2020250228152207.png)

添加凭据

![](assets/Pasted%20image%2020250228152245.png)

填写 ip地址、用户名、密码后保存即可

![](assets/Pasted%20image%2020250228152321.png)

完成后再去进行 [操作](#操作)
